# Regional Sales Analysis

## Project Overview

This project analyzes regional sales performance to identify trends, top-performing products, customer behavior, and revenue opportunities.

## Business Problem

The objective is to understand sales performance across different regions and provide actionable insights for business decision-making.

## Tools

- Python
- Pandas
- NumPy
- Matplotlib
- Power BI
- Excel

## Project Workflow

1. Data Cleaning
2. Data Exploration
4. KPI Calculation
5. Dashboard Creation
6. Business Insights

## Key Insights

- Top-performing regions
- Best-selling products
- Monthly sales trends
- Customer purchasing patterns
- Revenue contribution by category
